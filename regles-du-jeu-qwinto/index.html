Règles du jeu Qwinto

Les joueurs de Qwinto jouent tous en même temps, chacun essayant de remplir les lignes de ses feuilles de match avec des chiffres aussi rapidement que possible afin de marquer le plus de points possible.

Le joueur reçoit une feuille de match contenant trois rangées colorées de formes (principalement des cercles avec quelques pentagones); les rangées ne se chevauchent pas complètement, mais elles se chevauchent suffisamment pour créer cinq colonnes verticales de trois formes, avec un pentagone dans chaque rangée verticale.

Lors d'un tour, un joueur lance 1-3 dés, les dés étant les mêmes couleurs que les lignes: orange, jaune, violet. Chaque joueur peut placer n'importe quelle somme roulée dans une forme vide dans une rangée qui correspond à la couleur de l'un des dés. Deux règles doivent être suivies pour placer une somme dans une ligne:

  1. Tous les nombres dans une ligne doivent augmenter de gauche à droite.
  2. Aucun nombre ne peut être répété dans une colonne verticale.

Un joueur n’a pas à entrez le numéro dans une ligne, mais si le joueur actif, celui qui a lancé les dés, ne le fait pas, il doit alors cocher une case erronée sur sa feuille de match.

La partie continue jusqu'à ce que quelqu'un ait rempli deux lignes. leur feuille de match ou quelqu'un a compté quatre erreurs. Les joueurs puis marquer leurs points: Pour chaque colonne complétée sur leur feuille de match, un joueur marque des points égal au nombre dans le pentagone de cette colonne.

Pour chaque ligne complétée sur leur feuille de match, un joueur marque des points égal au nombre le plus à droite de cette ligne. . Pour chaque ligne incomplète de leur feuille de match, un joueur marque un point par numéro dans cette ligne. Pour chaque erreur, un joueur perd cinq points. Celui qui a la plus grande somme gagne!
