Règles du jeu Boggle

Boggle est un jeu de dés chronométré. Pendant 3 minutes, les joueurs doivent trouver les mots liés dans un plateau 4×4 de dés cubiques, chacun avec une lettre différente imprimée de chaque côté.

Les points sont attribués en fonction du nombre de lettres contenues dans le mot mais les mots doivent contenir au moins 3 lettres.

Les lettres doivent être adjacentes horizontalement, verticalement ou en diagonale. Les mots peuvent être singuliers ou pluriels et ne peuvent utiliser les cubes qu'une seule fois. Les mots doivent exister dans le dictionnaire.

Chaque joueur écrit tous ses mots sur un papier privé.
