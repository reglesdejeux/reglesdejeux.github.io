Règles du jeu Don’t Spill the Beans

Don't Spill the Beans est un jeu de dextérité destiné aux enfants de 5 ans et plus. Le contenu du jeu comprend un pot de haricots avec couvercle, un tas de minuscules haricots en plastique, une aire de jeu sur laquelle vous placez le pot de haricots et quatre plateaux pour contenir vos haricots.

Pour lancer le jeu, retirez le couvercle du pot de haricots. renversez les haricots sur votre surface de jeu et replacez le couvercle sur le pot. Placez le pot de haricots sur l'aire de jeu. Répartissez les haricots uniformément entre tous les joueurs. Le joueur qui reçoit le dernier haricot commence en premier.

Les joueurs déposent un haricot à tour de rôle dans le pot à haricots. Étant donné que le pot est suspendu légèrement au-dessus de l'aire de jeu par ses poignées, la balance se modifie avec chaque grain supplémentaire, provoquant ainsi un débordement. Lorsque les haricots tombent, le joueur qui a basculé le pot ajoute les haricots renversés à leur plateau.

Le jeu continue jusqu'à ce que quelqu'un manque de haricots. Le joueur qui manque de haricots en premier est le vainqueur.

C'est un excellent jeu. Il est recommandé à partir de 5 ans. Il y a énormément de "euh-oh" dans ce jeu qui le rend amusant pour les enfants. Quand vos enfants sont trop vieux pour renverser des haricots, essayez le jeu de société Topple Board pour une version plus adulte du même concept.
