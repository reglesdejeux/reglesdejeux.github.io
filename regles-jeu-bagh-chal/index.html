Les règles du jeu Bagh-Chal

3 min read

Avant de connaître les règles du jeu Tigres et des Chèvres, il faut savoir qu’il est le jeu national du Népal où il est connu sous le nom de Bagh-Chal.

Ce qu’il vous faut pour jouer Bagh-Chal

Le jeu Bagh-Chal est joué sur un plateau spécial de 5 x 5 points reliés orthogonalement pour former une grille. Il y a aussi des lignes diagonales d’un coin à l’autre et des lignes diagonales reliant le point médian de chaque côté aux points médians du côté adjacent.

Bagh Chal est un jeu à deux joueurs chaqu’un d’un côté et se joue avec deux pièces représentant un tigre et 20 pièces représentant des chèvres.

Quels sont les règles du jeu Bagh-Chal

Lancez une pièce de monnaie pour décider quel joueur jouera les tigres pour le premier match. Le tableau commence avec les 4 tigres sur les points d’angle et aucune chèvre sur le plateau.

Selon les règles du jeu Bagh-Chal, l’objectif des tigres est de capturer 5 chèvres. L’objectif des chèvres est d’ourler les tigres afin qu’ils ne puissent pas bouger.

Les joueurs se relaient pour jouer et les chèvres commencent. Le jeu comporte deux phases: dans la première phase, les chèvres sont placées sur le plateau. Une fois que toutes les chèvres ont été jouées au tableau, la deuxième phase commence dans laquelle les chèvres sont déplacées.

Quand c’est le tour du tigre, le joueur peut déplacer 1 tigre le long d’une ligne vers un point adjacent ou, si possible, le joueur peut sauter 1 tigre sur 1 chèvre adjacente et dans un point libre au-delà d’une ligne droite. Ce dernier mouvement capture la chèvre qui est retirée du plateau.

Quand c’est au tour des chèvres de jouer, si toutes les chèvres n’ont pas été placées sur le plateau, le joueur ne peut placer qu’une chèvre sur un point vacant du plateau. Si toutes les chèvres ont été placées sur le plateau, le joueur ne peut déplacer qu’une chèvre vers un point adjacent le long d’une ligne.

Les chèvres ne peuvent pas sauter ou prendre des tigres et ne peuvent bouger qu’une fois que les 20 chèvres ont été jouées sur le plateau. Les tigres ne peuvent pas sauter par-dessus d’autres tigres.

Pour éviter une impasse répétitive, on utilise généralement une règle du jeu Bagh-Chal supplémentaire qui dit qu’un joueur ne peut pas déplacer une pièce de sorte qu’une position des pièces sur le plateau soit répétée.

Comment gagner au jeu Bagh-Chal

En suivant les règles du jeu Bagh-Chal, la partie est gagnée par les tigres s’ils capturent 5 chèvres. La partie est gagnée par les chèvres si elles parviennent à empêcher un tigre de se déplacer.

Chaque tour devrait être composé de 2 parties, les joueurs échangeant les pièces pour le deuxième match. Si les chèvres ont gagné à chaque fois, le gagnant est la personne qui a le plus de chèvres à la fin de la partie quand elles jouent aux chèvres. Sinon, c’est un match nul.
