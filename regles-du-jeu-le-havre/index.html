Règles du jeu Le Havre

Le Havre est un jeu de plateau économique pour le développement de villes dans lequel les joueurs tentent d’acquérir et d’affiner les marchandises à expédier. Le jeu se déroule dans le port du Havre, une ville française.

À chaque tour, les marchandises sont d'abord ajoutées à divers espaces d'offre. Les marchandises servent soit à nourrir la communauté, à construire des bâtiments et des navires, soit à être transformées dans l’un des 33 bâtiments. Ici, les marchandises sont transformées en produits finis que les joueurs peuvent vendre.

Il existe 5 types de bâtiments: les bâtiments d'artisan, les bâtiments économiques, les bâtiments industriels, les bâtiments publics et les non-bâtiments. Ils constituent tous un bon investissement et une source de revenus, mais les navires fournissent la nourriture nécessaire pour nourrir les travailleurs.

Le bois, l’argile et le fer constituent des ressources, tandis que le poisson, les céréales et le bétail servent de nourriture.

le jeu se joue pour un nombre de tours déterminé et à la fin, le joueur ayant la plus grande richesse nette est le vainqueur.
